# Instruções para adicionar imagens de teste

Para testar as imagens dos produtos, adicione arquivos de imagem na pasta:
`static/images/produtos/`

Nomes sugeridos:
- filtro-oleo.jpg
- pastilha-freio.jpg  
- amortecedor.jpg
- correia-dentada.jpg
- vela-ignicao.jpg
- oleo-motor.jpg
- bateria.jpg
- filtro-ar.jpg
- radiador.jpg
- kit-embreagem.jpg

Ou use a funcionalidade de upload da API para enviar imagens.
